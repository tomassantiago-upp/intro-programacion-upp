# intro-programacion-upp
Repositorio para la clase de introducción a la programación
⚡ Protocolo Stark: Introducción a la Programación

Bienvenidos al repositorio oficial del curso. Aquí encontrarán todo el código fuente, ejercicios y simulaciones que usaremos en clase.
No memorizaremos código; aprenderemos a pensar, a diseñar arquitecturas lógicas y a utilizar la Inteligencia Artificial como nuestro copiloto de desarrollo.

🚀 ¿Cómo usar este repositorio?

Los archivos de código de este curso están en formato Notebook (.ipynb). Esto significa que puedes abrirlos directamente en tu navegador usando Google Colab, sin instalar nada en tu computadora.

Instrucciones para alumnos:
1. Navega a la carpeta del Bloque actual.
2. Haz clic en el archivo de la clase (ej. 01_Variables.ipynb).
3. Verás un botón azul que dice "Open in Colab". Haz clic ahí.
4. Una vez en Colab, ve a Archivo > Guardar una copia en Drive para que puedas editar el código y hacer tus propios apuntes.

🤝 Protocolos de Trabajo (Reglas de Clase)
1. Prohibido Sufrir en Silencio: Si te bloqueas, pregunta al equipo, a la IA o al profesor.
2. Copiar sin entender es veneno: Puedes usar ChatGPT/Gemini para generar código, pero debes ser capaz de explicar línea por línea cómo funciona. La IA escribe, tú auditas.
3. Fail Fast: Equivócate rápido y barato. Un error en pantalla no es un fracaso, es información."No eres un estudiante. Eres un Arquitecto de Soluciones."
